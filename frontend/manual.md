## 1. 系统概述

零售管理运营平台是一套面向授权体验店的全链路运营管理系统，覆盖**销售管理、库存监控、价格管控、店员提成、会员运营、知识库**等核心功能。

### 技术规格

| 项目 | 说明 |
|------|------|
| 访问地址 | `http://localhost:9527` |
| 支持浏览器 | Chrome / Edge / Safari（推荐 Chrome） |
| 移动端 | 支持手机浏览器 + PWA 安装到桌面 |
| 数据存储 | SQLite 本地数据库（`data/rm_ops.db`） |
| 会话时长 | 登录后 72 小时有效 |

### 角色与权限

| 角色 | 权限范围 |
|------|---------|
| **管理员 (admin)** | 全部功能 + 用户管理 + 系统配置 |
| **店长 (manager)** | 本店数据 + 通用的提成/知识库/协作功能 |
| **店员 (staff)** | 有限访问（考勤打卡、知识库查看等） |

## 5. 💰 销售进度

**访问路径**：左侧导航 → 💰 销售进度

### 视图切换

页面顶部有三个 Tab 可切换：

| Tab | 内容 |
|-----|------|
| **月度报表** | 当月各门店的手机销售额、台量、NCME、配件数据 |
| **对比分析** | 环比（vs上月）/ 同比（vs去年同月）对比，含 Chart.js 柱状图 |
| **门店排名** | 按完成率/台量/销售额等指标排名，含 🥇🥈🥉 奖牌 |

### 操作指南

1. **选择月份**：顶部选择器切换查看不同月份数据
2. **切换门店**：管理员可下拉筛选门店，店长只能看本店
3. **导出数据**：
   - 点击 **导出 CSV**：下载为 Excel 可打开的 CSV 文件（UTF-8 编码）
   - 点击 **导出 Excel**：下载为 .xlsx 格式

### 指标说明

| 指标 | 含义 |
|------|------|
| 手机销售额 | S26/Fold7/Flip7/W26 等机型销售额合计 |
| 台量 | 销售手机台数 |
| NCME 销售额 | NCME（New Computing Mobile Experience）类产品销售额 |
| 配件销售额 | 手机壳/贴膜/耳机/充电器等配件销售额 |
| 完成率 | 销售额 ÷ 月度目标 × 100% |

## 6. 📦 库存监控

**访问路径**：左侧导航 → 📦 库存监控

### 功能说明

| 区域 | 说明 |
|------|------|
| **库存列表** | 查看各门店/各机型的当前库存数量和状态 |
| **库存预警** | 可销天数不足时自动标红/标黄 |
| **补货建议** | 智能分析销售速度，给出补货优先级建议 |

### 库存状态

| 状态 | 颜色 | 含义 | 建议 |
|------|------|------|------|
| 🟢 **正常** | 绿色 | 库存充足 | 维持现状 |
| 🟡 **预警** | 黄色 | 可销 < 7 天 | 关注并准备补货 |
| 🔴 **紧急** | 红色 | 可销 < 3 天 | 立即下单补货 |

### 操作指南

1. 选择**门店**和**机型系列**筛选
2. 查看"可销天数"列判断库存健康度
3. 点击**补货建议** Tab 获取智能推荐
4. 库存数据支持从亿博士（eBoss）系统自动同步

## 8. 💵 店员提成

**访问路径**：左侧导航 → 💵 店员提成

### 页面结构

5 个 Tab 切换不同视图：

| Tab | 内容 |
|-----|------|
| **店员列表** | 按门店查看店员底薪、销量、提成、预计收入 |
| **提成概览** | Chart.js 环形图：手机/NCME/配件/回收提成分布 + 门店排行 |
| **销售排行** | 🥇🥈🥉 销冠榜单，支持按指标排序 |
| **提成规则** | S26/Fold7/Flip7/W26/NCME/配件/回收各品类提成标准 |
| **活动政策** | 当前活动政策、培训通知展示 |

### 操作指南

1. 选择**门店**和**月份**筛选
2. 点击店员行查看**详情弹窗**（含进度条、提成明细、销售数据）
3. 在店员列表页可**新增店员**或**同步数据**

### 提成计算公式

```
预计收入 = 底薪 + (手机销量 × 手机提成单价)
         + (NCME销售额 × NCME提成比例)
         + (配件销售额 × 配件提成比例)
         + 回收提成
```

### 能力清单

| 能力 | 示例提问 |
|------|---------|
| 📊 **销售查询** | "万象城店本月卖了多少台 S26？" |

## 17. 常见问题

### Q1：登录失败怎么办？

- 检查账号密码是否正确（区分大小写）
- 确认服务是否正常运行（浏览器访问 `http://localhost:9527`）
- 如忘记密码，联系管理员重置

### Q2：页面显示异常/排版错乱？

**最快解决方式**：按 `Ctrl + Shift + R`（硬刷新）跳过浏览器缓存。

### Q3：数据不更新？

- 销售和库存数据来源于亿博士（eBoss）系统
- 管理员可在后台管理 → 数据同步 → 手动触发同步
- 同步频率默认每小时一次

### Q4：AI 助手不回答？

- 确认 AI API 密钥已配置（管理员操作）
- 刷新页面后重试
- 如果回答中包含技术标记（`<| DSML |`），硬刷新页面即可

### Q5：如何在手机上使用？

1. 手机浏览器访问 `http://<电脑IP>:9527`（需同一网络）
2. 浏览器菜单 → **添加到主屏幕** → 安装为 PWA 应用
3. 安装后可在桌面像 App 一样打开，支持离线查看已缓存的页面

### Q6：服务突然不可用？

**Windows**：
```powershell
# 重新启动服务
cd D:\Workbuudy\rm-ops
python main.py
```

**检查点**：
- 确认 Python 进程未意外关闭
- 确认 9527 端口未被占用
- 检查 `data/rm_ops.db` 文件是否存在且完好

### Q7：如何修改密码？

1. 登录后点击右上角头像区域
2. 选择"修改密码"
3. 输入旧密码和新密码
